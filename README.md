# todos-mobx

A to-do implementation using [MobX](https://github.com/mobxjs/mobx) and [mobx-vue](https://github.com/mobxjs/mobx-vue) on Vue.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
